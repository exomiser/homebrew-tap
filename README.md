# Exomiser Tap

## How do I install these formulae?

`brew install exomiser/tap/<formula>`

Or `brew tap exomiser/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "exomiser/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
